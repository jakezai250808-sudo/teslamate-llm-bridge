### How to Sign | 如何签署

When you open a Pull Request, the **cla-assistant bot** will post a comment asking you to sign this CLA. Click the link in the comment and follow the instructions. The bot will automatically mark your PR once you have signed.

提交 Pull Request 时，**cla-assistant 机器人**会在评论区发送签署请求。点击评论中的链接并按指引完成签署，机器人将自动标记您的 PR 状态。

> **Note / 注意**: The CLA signature is recorded once per GitHub account. You only need to sign once for all future contributions to this repository.
> CLA 签署按 GitHub 账号记录，同一账号只需签署一次，后续所有贡献均适用。
