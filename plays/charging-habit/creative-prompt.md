# charging-habit 创作引导 prompt（探索层）

> 两阶段漏斗的第一阶段：用户拿到 play JSON 结果后，用自己的生图模型
> （GPT Image / 豆包生图 / 即梦）自由创作分享图。本文件是推荐 prompt 模板 ——
> 好 prompt 决定好图的下限。爆款设计经筛选后由维护者固化成 `card.svg.tmpl`
> （第二阶段，见 `docs/play-creative-pipeline.md`）。

## 给 LLM 平台的使用方式

LLM（ChatGPT / 豆包 bot）调用 `runPlay(charging-habit)` 拿到 JSON 后，
应主动建议用户：「要把这个充电人格做成一张分享图吗？」用户同意后，把下面模板的
`{占位符}` 用 JSON 里的真实值填充，调用生图能力。

## 推荐 prompt 模板 v2（深夜实景风·带刻度条形图·GPT标杆对齐版）

> v2 相比 v1 的改进：① 三轴条形图带 0/25/50/75/100 刻度轴；② 每条指标下有一行人格化副文案；
> ③ 主体车辆融入深夜充电站实景而非浮贴背景；④ 人格码描述与字号严格分句（修复 v1 尺寸词入画 bug）。

```text
写实深夜质感信息图，小红书爆款竖版，深海蓝黑底。黑色特斯拉 Model 3 停超充站，已插充电线，积水地面反光，琥珀橙绿混合光晕，真实夜景氛围，车与场景自然融合。

版面从上到下六个层次：

【第一层 顶部徽章】
单行小字徽章：充电人格测试 Vol.1（带小盾牌图标）

【第二层 标题组】
大粗体中文：你是一个典型的{persona.name}
中号中文：{persona.desc}

【第三层 人格码 + 自嘲标签】
- 用超大字号显示「{code}」这几个字母（只有人格码字母本身，不加其他字符），琥珀橙配色，横排居中或偏左
- 旁边标「人格码」小字标签
- 发光边框胶囊：「{persona.tag}」

【第四层 三轴条形图——三条各自独占完整行高】

第一行完整条：
  标题「电量焦虑指数」+ 右侧数值「{anxiety}」
  橙红渐变横条，填充约 {anxiety}% 宽度
  条下方刻度标签：0    25    50    75    100
  下方副文案一行：「电门雷达拉满 焦虑早于低电量提示响起」

第二行完整条：
  标题「充满率」+ 右侧「{full_pct}%」
  琥珀橙横条填 {full_pct}% 宽
  刻度：0    25    50    75    100
  副文案：「七成充满 谨慎派的安心底线」

第三行完整条：
  标题「家充占比」+ 右侧「{home_pct}%」
  充电绿横条填 {home_pct}% 宽
  刻度：0    25    50    75    100
  副文案：「家即充电站 从不在外漂泊的能量」

【第五层 数字看板 3 列等宽】
列1：闪电图标 / 充电次数 / 大字「{charge_count}」
列2：电池图标 / 总充入 / 大字「{total_kwh}kWh」
列3：房子图标 / 家充占比 / 大字「{home_pct}%」
每列上图标，中小标签，下大数字，三列等宽横排。

【第六层 底部互动】
大字：「你是哪种充电人格？评论区聊聊」
小字：「数据来自真实充电记录，仅供娱乐参考」

全局：中文排版，信息密度高但各层有间距不挤压，不出现车牌地名充电桩编号。
```

> 提分记录：对标 GPT FNLE 卡，5 张样本最佳 24/25（2026-06-10）

## 推荐 prompt 模板 v1（深色电流风示例，已被 v2 超越，保留供参考）

```text
用我的真实充电数据画一张小红书风格的充电人格分享图（生成图片）：

【数据】来自我的特斯拉真实充电记录
- 充电人格码：{code}「{persona.name}」
- 一句话画像：{persona.desc}
- 自嘲标签：{persona.tag}
- 三轴：电量焦虑 {anxiety}/100 · 充满率 {full_pct}% · 家/固桩占比 {home_pct}%
- 本期：{charge_count} 次充电 · 平均起充 {avg_start_soc}% · 总充入 {total_kwh} kWh

【设计要求】
- 竖版 3:4 小红书爆款信息图
- 深色电流感主题：深海蓝黑底色、琥珀橙/充电绿高亮，像电流在深夜流动
- 超大人格码 {code} 视觉锤，琥珀橙配色（注意：描述字号时不要把字号数字写入可见文字）
- 标题「充电人格测试：你是一个典型的{persona.name}」
- 三轴条形图：电量焦虑用橙红渐变、充满率用琥珀橙、家充占比用蓝绿色
- 数字看板展示真实数字（数字必须准确，不要编造）
- {persona.tag} 做成发光徽章效果
- 底部互动钩子「你是哪种充电人格？评论区聊聊」
  + 小字「数据来自真实充电记录，仅供娱乐参考」
- 中文排版，信息密度高但层次清晰
- 不要出现真实车牌 / 具体地名 / 充电桩编号
```

## 变体方向（用户可追加一句话切换）

| 追加指令 | 效果 |
|---|---|
| 「换成手账风」 | 浅色可爱风，充电桩插头贴纸装饰 |
| 「做成 A/B 对比版，我和另一辆车各一个人格」 | 两台车充电风格对比（多车家庭传播力强） |
| 「赛博朋克霓虹风」 | 高饱和蓝紫+橙，充电站夜景氛围 |
| 「极简风，只要人格码和一句话」 | 适合做头像/锁屏壁纸 |

## 安全注意

- prompt 中不放 VIN / 车牌 / 精确充电地址；数据已是聚合统计无 GPS 坐标
- 提醒用户：生成图含「仅供娱乐参考」免责字样再发布
