# charging-habit 创作引导 prompt（探索层）

> 两阶段漏斗的第一阶段：用户拿到 play JSON 结果后，用自己的生图模型
> （GPT Image / 豆包生图 / 即梦）自由创作分享图。本文件是推荐 prompt 模板 ——
> 好 prompt 决定好图的下限。爆款设计经筛选后由维护者固化成 `card.svg.tmpl`
> （第二阶段，见 `docs/play-creative-pipeline.md`）。

## 给 LLM 平台的使用方式

LLM（ChatGPT / 豆包 bot）调用 `runPlay(charging-habit)` 拿到 JSON 后，
应主动建议用户：「要把这个充电人格做成一张分享图吗？」用户同意后，把下面模板的
`{占位符}` 用 JSON 里的真实值填充，调用生图能力。

## 推荐 prompt 模板 v1（深色电流风示例）

```text
用我的真实充电数据画一张小红书风格的充电人格分享图（生成图片）：

【数据】来自我的特斯拉真实充电记录
- 充电人格码：{code}「{persona.name}」
- 一句话画像：{persona.desc}
- 自嘲标签：{persona.tag}
- 三轴：电量焦虑 {anxiety}/100 · 充满率 {full_pct}% · 家/固桩占比 {home_pct}%
- 本期：{charge_count} 次充电 · 平均起充 {avg_start_soc}% · 总充入 {total_kwh} kWh

【设计要求】
- 竖版 3:4 小红书爆款信息图
- 深色电流感主题：深海蓝黑底色、琥珀橙/充电绿高亮，像电流在深夜流动
- 超大人格码 {code} 视觉锤（180px），琥珀橙配色
- 标题「充电人格测试：你是一个典型的{persona.name}」
- 三轴条形图：电量焦虑用橙红渐变、充满率用琥珀橙、家充占比用蓝绿色
- 数字看板展示真实数字（数字必须准确，不要编造）
- {persona.tag} 做成发光徽章效果
- 底部互动钩子「你是哪种充电人格？评论区聊聊」
  + 小字「数据来自真实充电记录，仅供娱乐参考」
- 中文排版，信息密度高但层次清晰
- 不要出现真实车牌 / 具体地名 / 充电桩编号
```

## 变体方向（用户可追加一句话切换）

| 追加指令 | 效果 |
|---|---|
| 「换成手账风」 | 浅色可爱风，充电桩插头贴纸装饰 |
| 「做成 A/B 对比版，我和另一辆车各一个人格」 | 两台车充电风格对比（多车家庭传播力强） |
| 「赛博朋克霓虹风」 | 高饱和蓝紫+橙，充电站夜景氛围 |
| 「极简风，只要人格码和一句话」 | 适合做头像/锁屏壁纸 |

## 安全注意

- prompt 中不放 VIN / 车牌 / 精确充电地址；数据已是聚合统计无 GPS 坐标
- 提醒用户：生成图含「仅供娱乐参考」免责字样再发布
