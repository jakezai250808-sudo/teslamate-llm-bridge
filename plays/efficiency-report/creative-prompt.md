# efficiency-report 创作引导 prompt（探索层）

> 两阶段漏斗的第一阶段。用户拿到 play JSON 后，在自己的生图模型
> 自由创作分享图。本文件是推荐 prompt 模板。
> 爆款经筛选后由维护者固化成 `card.svg.tmpl`（第二阶段）。

## 给 LLM 平台的使用方式

调用 `runPlay(efficiency-report)` 拿到 JSON 后，主动建议：
「要把你的能效报告做成一张可以晒的图吗？」用户同意后用下面模板。

## 推荐 prompt 模板 v2（Seedream 优化版，2026-06-10 实测 24/25 分）

> 对应 Seedream doubao-seedream-4-0 模型，竖版 1536x2048 尺寸。
> 修复已知 bug：进度条右端「超耗」（非「超省」）；底部互动句留足内边距不裁切。

```text
质感科技感竖版信息卡片，高速公路夜景长曝光实拍质感背景，城市灯光轨迹虚化，深蓝黑底色，霓虹橙蓝渐变光晕。一辆黑色 Model 3 从画面右下角驶向左侧，车灯拖出橙色光轨，车身融入夜景背景而非贴图。
布局从上到下：
第一行：「⚡ 能效报告 · 近 {window_days} 天」蓝色胶囊徽章，旁边副标题「你的电门比发电厂还忙」。
第二行：左侧超大橙红数字「{actual_wh_km} Wh/km」视觉锤，小标签「实际能效」；右侧大字「{grade_key}」等级橙红光晕，下方「{grade_info.title}」，「{grade_info.tag}」橙色发光徽章。
第三行：三列数据卡，每列含图标+小标签+大数字+一行副文案。第一列「⚡ 效率比 / {efficiency_ratio}%」副文案「{grade_info.desc}」；第二列「💰 每百公里 / +¥{delta_cost_per_100km}元」副文案「多烧的钱 都换成了激情」；第三列「🔋 本期多花 / +¥{period_delta_cost}元」副文案「一杯奶茶的代价」（注：period_delta_cost 为负数时改为「省了」并调整副文案）。
第四行：能效进度条，从深绿色渐变到深红色，左端标注「超省」，右端标注「超耗」（右端必须是「超耗」），色条下方标注五个刻度 0、25、50、75、100，白色虚线在中点 50 处垂直穿越色条作基准线，橙红色向上三角形标记在实际位置并标注「你在这」，整体副文案「多花的那部分，都充进了加速里」。
第五行底部：完整显示互动句「你的 Tesla 每公里花多少电？评论区晒 👇」，小字「数据来自真实行程，仅供参考」，底部留足内边距确保文字完整不被裁切。整体中文字体清晰，无车牌无地址。
```

> 提分记录：对标 GPT FNLE 卡，3 张样本最佳 24/25（2026-06-10）

## 推荐 prompt 模板 v1（原始版，已被 v2 取代）

```text
用我的特斯拉真实能效数据画一张小红书风格分享图：

【数据】来自真实行车记录（近 {window_days} 天）
- 实际能效：{actual_wh_km} Wh/km（车型基准 {baseline} Wh/km）
- 效率等级：{grade_key} 级「{grade_info.title}」
- 等级标签：{grade_info.tag}
- 一句话画像：{grade_info.desc}
- 每百公里差价：{delta_cost_per_100km} 元（正数多花，负数省）
- 本期合计多花/省：{period_delta_cost} 元
- 共充电 {charge_sessions} 次，补能 {total_kwh_charged} kWh

【设计要求】
- 竖版 3:4 小红书信息图，深色科技感（深蓝黑 #0d1b2a 底）
- 超大实际能效数字「{actual_wh_km}」视觉锤，等级主色高亮
- 核心叙事：本期多花/省了多少元电费（用大字说人话，把 Wh/km 翻译成金额）
- 能效进度条：左端「超省」→ 右端「超耗」，基准线用虚线竖穿中点，
  实际位置用等级色彩标注，一眼看出偏哪侧
- 等级大字 {grade_key} + 等级名「{grade_info.title}」配等级主色光晕
- {grade_info.tag} 做成发光徽章
- 底部互动钩子：「你的 Tesla 每公里花多少电？评论区晒」
  + 小字「数据来自真实行程，仅供参考」
- 中文排版，不出现真实车牌或精确地址
```

## 变体方向

| 追加指令 | 效果 |
|---|---|
| 「换成极简风，只要等级和金额两行字」 | 适合做 9 宫格第一图，冲击力强 |
| 「做成 A/B 对比版，两种驾驶风格」 | 同车主不同季节的能效对比，互猜钩子 |
| 「加上月度充电次数和里程曲线」 | 数据更丰富，适合技术派晒图 |
| 「赛博朋克风，仪表盘感」 | 高对比霓虹，适合 F 级用电大户自嘲狂欢 |

## 安全注意

- prompt 中不放 VIN / 车牌 / 精确地址
- 生成图含「仅供参考」免责字样再发布
- `period_delta_cost` 为负数时措辞为「省了」而非「多花」，需在 prompt 里判断
